Importar configuración de Steam

Selecciona tu carpeta de Steam. El generador detectará automáticamente todas las cuentas con datos de CS2 y leerá tus archivos .vcfg sin necesidad de subirlos a ningún servidor.

Navegador no compatible.
La lectura de archivos locales requiere Chrome 86+ o Edge 86+. Firefox no soporta la File System Access API. Abre esta página en Chrome o Edge.

Exportar Autoexec

Elige dónde guardar el archivo generado.

Escribir directo a game/csgo/cfg/
Usa showSaveFilePicker() para guardar el archivo exactamente donde CS2 lo necesita. Requiere que Steam esté instalado en la ruta estándar o que selecciones la carpeta manualmente.
Descargar como archivo
Descarga autoexec.cfg a tu carpeta de Descargas. Luego muévelo manualmente a game/csgo/cfg/.
Ruta destino detectada
C:\Program Files (x86)\Steam\steamapps\common\Counter-Strike Global Offensive\game\csgo\cfg → autoexec.cfg
Asegúrate de que tu launch option incluya +exec autoexec para que CS2 cargue el archivo al arrancar.
¡Autoexec exportado!
El archivo fue guardado correctamente. Reinicia CS2 o escribe exec autoexec en la consola para cargarlo.
exec autoexec
